# Security Policy

## Current Status

AI-Core is an independent open source project built and maintained 
by one developer. Formal security testing has not been completed yet.

This is real work in active development — not a production hardened 
system. Use accordingly.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v4.x    | ✅ Active development |
| v3.x    | ❌ No longer maintained |
| < v3.0  | ❌ No longer maintained |

## Reporting a Vulnerability

If you discover a security vulnerability please report it directly:

📧 comanderanch@gmail.com

I am one independent developer with two AI collaborators 
(Claude Browser and Claude CLI — Anthropic).

I will respond as quickly as possible, acknowledge the report 
honestly, and address it to the best of my ability.

No bounty program exists yet. Just honest open source work 
and a genuine commitment to fixing real problems.
